# 🧰 在线工具箱

基于 React + Vite 的在线工具集合，所有数据处理均在浏览器本地完成，无需上传。

## ✨ 特性

- **98 个工具** · **13 个分类**
- 🔐 加密加密 | 🔣 编码解码 | 📝 格式化 | 🎲 生成器 | 📄 文本处理 | 🔄 转换器
- 🌐 网络工具 | 🖼️ 图片工具 | 📐 数学计算 | 📅 日期时间 | 📋 快捷键 | 🔍 查询 | 🎮 小游戏
- 🌙 暗色模式 · 🌍 中英文切换 · 📱 移动端适配
- 🔒 所有数据本地处理，隐私安全

## 🚀 本地运行

```bash
npm install
npm run dev -- --host
```

## 📦 技术栈

React 19 · Vite 8 · CryptoJS · QRCode · i18next
